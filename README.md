# OCR solution based on EasyOCR (https://github.com/JaidedAI/EasyOCR.git)

## Installation

```
git clone https://github.com/JaidedAI/EasyOCR.git
cd EasyOCR.git
pip install .
pip install onnx onnxruntime

or

pip install -r requirements.txt
```
